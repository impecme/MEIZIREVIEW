# Markdown基础用法

*代表拓展语法, 有些编辑器用不了,Typora可以用

```
[TOC]生成目录	
```



[TOC]



## 0. 写Markdown的第0步

全角标点不被认可，写Markdown时全都要使用半角标点, 每个半角标点后需要加上后置空格, 符合英文标点的书写规范, 同时也更加美观. 

在微软输入法中按住 `Ctrl`  + `.(句号, 不是点)` , 可以切换标点的全角与半角, 这样可以实现 `中文输入` + `半角标点` 

## 1. 标题 [数个`#`+` (空格)` 前置]

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

标题会在目录和大纲分级显示, 可以跳转
在Typora中建议开启 `严格模式` , 即不应该为`#标题` , 应为`# 标题` . 
应该要手动补上空格, 使得Markdown语法在其他的文本编辑器上兼容. 

## 2. 强调(加粗) [用 `**` 或者 `__` 包围]

```
**dz真帅**
__dz真帅__
```

或者选中要强调的文字按下 `Ctrl` + `B` . 
如: **dz真帅** 

## 3. 斜体 [使用 `*` 或者 `_` 包围]

```
*dz真帅*
_dz真帅-
```

或者选中要设置为斜体的文字按下 `Ctrl` +  `I` 

如: *dz真帅* 

## 4. 斜体并强调 [使用 `***` 或者 `___` 包围]

```
***dz真帅***
___dz真帅___
```

如: ***dz真帅*** 

## 5. 删除线 [使用 `~~` 包围]

```
~~dz真帅~~
```

如: ~~dz真帅~~ 

## 6. *高亮 [使用 `==` 包围]

**(注意: 此为拓展语法)** 

```
==dz真帅==
```

如: ==dz真帅==

## 7. 代码 [使用 "`"  包围] 

```
`dz真帅`
```

## 8. 代码块 [按下三个 "`" 并敲下回车或者快捷键 Ctrl + Shift + K]

~~~c
```
//输入需要的代码
#include <stdio,h>
int main()
{
	return 0;
}
~~~

在 Typora 的偏好设置中勾选显示行数的选项. 

代码块中可以选择语言, 会根据不同的语言来高亮各个语句, 在选择语言后,  ``` 会变为 ```` + 对应语言.

## 9. 引用 [`>` + 空格 前置]

``` 
> dz真帅
> > dz真帅
```

**引用是可以嵌套的! **

如: 

> dz真帅

> > dz真帅

 ## 10. 无序列表 [ `-` 或者 `+` 或者 `*` + 空格 前置]

```
- dz真帅1
+ dz真帅1
* dz真帅1
```

如: 

- dz真帅1
+ dz真帅1

(回车)

* dz真帅1
  * (Tab)

Typora中Tab可以缩进一级, 回车可以退回一级

## 11. 有序列表 [`数字` + `.` +空格 前置]

```
我想说三件事:
1. dz真帅
2. dz真特么帅
3. dz是真特么的帅
```

如: 

我想说三件事:
1. dz真帅
2. dz真特么帅
3. dz是真特么的帅

## 12. *上标 [使用 `^` 包围]

**(注意: 此为拓展语法)** 

```
2^2^=4
```

如: 2^2^=4

## 13. *下标 [使用 `~` 包围]

**(注意: 此为拓展语法)** 

```
H~2~O
```

如: H~2~O

## 14. *注释 [使用 `[^数字]` 后置]

```
> dz真帅[^1]
(注释1)

[^1]dz真帅
(放在文本末)
```

如: 

> dz真帅[^1] 

[^1] dz真帅

## 15.链接 [常用 `[]` + `()` 分别包围文本与链接]

```
[dz真帅](https://github.com)
[基础教程: 12. 下标](#12. *上标 [使用 `^` 包围])
(#后跟文内链接)
```

支持网页链接和文内链接跳转, 按住 `Ctrl` + `单击鼠标左键` 即可跳转. 

如: 

 [dz真帅](https://github.com)
[基础教程: 12. 下标](#12. *上标 [使用 `^` 包围])

## 16. 任务列表 [`- [空格]` +空格 前置]

```
Todolist:
- [ ] 打视频
- [ ] 刷b站
- [ ] 写代码
- [ ] 看文献
```

如: 

Todolist:

- [ ] 打视频
- [ ] 刷b站
- [ ] 写代码
- [ ] 看文献

## 17.表格 [使用 `|` 绘制表格边框]

```
| dz | dz | dz |
| :- | :-: | -: |(冒号位置代表左对齐, 居中, 右对齐)
| 好帅 | 好帅 | 好帅 |
```

如:

| dz | dz | dz |
| :- | :-: | -: |
| 好帅 | 好帅 | 好帅 |
